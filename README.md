## 📌 Overview
S.P.O.T is a portable, real-time surveillance and protection system concept built in 24 hours for the EASE'22 hackathon, taking first place. It was designed to extend security coverage to up to 1 km while cutting down the manual labour typically required for monitoring — combining live alerting, secure logging, and accessible cloud-based analytics into a single deployable unit.

## 🔍 Key Features
- **Live monitoring & alerts** — real-time surveillance feed with automated alerting
- **Blockchain-secured logs** — tamper-resistant activity logging
- **Data analytics** — Python, SQL, and Power BI for surveillance data insights
- **Flask web interface** — accessible control and monitoring dashboard
- **Cloud-enabled deployment** — scalable, remote access from anywhere

## 🛠️ Tools & Technologies
Python, SQL, Power BI, Flask, Blockchain (for log integrity)
